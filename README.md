# Midterm Programming Project |
## THE BANK JOB: ATM Simulation System

### Scenario | 🏦 Designing Your Own Automated Teller Machine
Imagine you were hired by a local bank in the Philippines to help develop a
**_simulation of an Automated Teller Machine (ATM) system._**

Design and implement a comprehensive Automated Teller Machine (ATM)
simulation that mirrors real-world banking operations. This project serves as a
capstone for fundamental C++ programming concepts, requiring your group to
create a dual-interface system with distinct functionality for administrators and
clients.

**This ATM will be used by different types of cardholders and must simulate real-world
banking transactions.**
